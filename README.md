# OpsGuard AI

Secure AI Document Review Platform.

OpsGuard AI is a learning and portfolio project focused on building a secure, testable AI-powered document review platform step by step.

## Current Scope

Step 1 initializes the monorepo with:

- FastAPI backend
- Next.js frontend
- Basic documentation structure
- Data folders for raw, synthetic, and evaluation documents

No AI, RAG, authentication, or database logic yet.
